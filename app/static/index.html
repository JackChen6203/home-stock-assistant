手機可直接操作的 Home Stock Assistant

Home Stock Assistant

這是一個可在手機瀏覽器直接使用的購物清單 MVP。預設連到你目前的 Render 服務，不需要本機架設，也不需要另外跑 Flutter。

連線設定

預設會使用目前頁面的網址作為 API 基礎路徑。若你想改成其他 Render 網址，可以直接覆蓋。

建議保留同網域，這樣手機打開頁面後就能直接操作。
準備連線中。

註冊 / 登入

先註冊一次，再登入取得 token。登入後就能新增項目、建立家庭與刷新清單。

家庭管理

建立家庭後才能把家用清單分給多人一起使用。

購物清單

加入個人或家用清單，或直接用購買按鈕模擬扣除。語音項目也可直接寫入清單。

目前清單

登入後按重新載入清單，就能看到個人與家用待購項目。
